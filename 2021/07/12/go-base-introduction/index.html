Advantages of Go

  • Simple syntax
  • Easy to write concurrent programs
  • Compiled language
  • Fast compilation
  • Static linking
  • Go Tooling
  • Garbage collection
  • Simple language specification

代码解析

  • package main: 每个代码必须以 package 开头，包提供代码划分(code compartmentalisation)和可重用性(reusability)。
  • import "fmt": 导入包 ，fmt 将文本打印到标准输出。
  • func main(): 程序从主函数开始。
  • fmt.Println("Hello World"): 将文本写入标准输出。
